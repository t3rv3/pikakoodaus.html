Viisi eri koodikieltä

Python

Python on yleiskäyttöinen ohjelmointikieli, jota käytetään laajasti.

JavaScript

JavaScript on yleisesti käytetty ohjelmointikieli, jota käytetään verkkosivujen ja sovellusten kehittämisessä.

Java

Java on suosittu ja monikäyttöinen ohjelmointikieli, jota käytetään laajasti erilaisissa sovelluksissa.

C++

C++ on tehokas ja monipuolinen ohjelmointikieli, jota käytetään esimerkiksi pelien ja järjestelmäohjelmistojen kehittämisessä.

Ruby

Ruby on dynaaminen ohjelmointikieli, jota käytetään erityisesti web-sovellusten kehittämisessä.
